1. Open the terminal in the directory.
2. Activate the environment.
3. pip install -r requirements.txt.
4. Update the openai api key in .env file. 
5. Change the api key and secret key in the .env file.
6. Run the create_assistant.py file buy typing in terminal "python create_assistant.py".
7. Run main.py file by typing in terminal "python main.py".
8. Begin the conversation with Hi.
9. After completing the conversation make sure to type exit in the interface and don't use force close of terminal.
