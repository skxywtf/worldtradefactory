1. Open the terminal in the mod_latest directory.
2. pip install -r requirements.txt.
3. Change the api key and secret key in the chat_session.py file.
4. Run the create_assistant.py file buy typing in terminal "python create_assistant.py".
5. Copy the assistant id and thread id that is generated.
6. Paste the assistant id in .env file.
7. Paste the thread id in data.json file.
8. Run main.py file by typing in terminal "python main.py".
9. Begin the conversation with Hi.
10. After completing the conversation make sure to type exit in the interface and don't use force close of terminal.
